# Kivy Music Player
Player de musica desenvolvido com Python, Kivy e KivyMD



## Pré Instalação

* instalar as dependencias do kivy >> [Kivy](https://kivy.org/doc/stable/gettingstarted/installation.html)
* instalar as dependencias do pipenv >> [LINK](https://pipenv.readthedocs.io/en/latest/install/#installing-pipenv)

## Instalação
 
### Desenvolvimento
```
git clone https://github.com/JuacyWillian/kivy-player.git
cd kivy-player

pipenv install --dev
```

### Usuario
```
git clone https://github.com/JuacyWillian/kivy-player.git
cd kivy-player

pipenv install
```

### Rodando o player
```
pipenv run python main.py
```
